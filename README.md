# 📊 Sales KPI Dashboard

## 📌 Project Overview
This project is a Sales KPI Dashboard created using Microsoft Excel.  
It helps in tracking and analyzing key business performance metrics like sales, profit, orders, and regional performance using interactive charts and visualizations.

---

## 🎯 Objective
- To analyze sales performance using KPIs
- To visualize business data in a simple and effective way
- To help in data-driven decision making

---

## 📊 Key Performance Indicators (KPIs)
- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Orders
- 👨‍💼 Sales Performance
- 🌍 Region-wise Sales Analysis
- 🛍 Product Category Insights

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization Techniques
